Web pictures

Social media

Web pictures alberga un sistema de registro de usuarios con email. Estos gestionan un perfil propio y pueden enviarse mensajes entre si de forma asíncrona. Todas las publicaciones y perfiles se pueden crear, borrar y editar en tiempo real. Cuenta con un botón like/dislike el cual funciona con un formulario, cuenta los 'likes' y los registra en la base de datos. La web implementa un sistema de amigos para relacionar los usuarios que funciona desde el backend.

La app ha sido desarrollada en Django-2.2, se apolla en Pillow-7.0.0 para gestionar las imágenes, y django-crispy-forms-1.9, para la estética de los formularios. Utiliza la librería django-cleanup-4.0.0 y así las imágenes borradas desaparecen por completo de la app.

Proyecto Url-shortener

Recorta urls

Esta app permite al usuario introducir una dirección web en su página principal, donde podrá transformarla en un código de cinco dígitos que funcionara como la url original.

La aplicación programada en Django (v 3.0.2) guarda las urls y los códigos en una base de datos SQL y si el usuario lo desea, puede registrarse y administrar su página de perfil con las urls y códigos que él mismo haya almacenado. La app se puede gestionar a través del admin panel. Para el desarrollo se ha usado TDD y plantillas Bootstrap para el frontend.

Se puede ejecutar localmente en cualquier ordenador a través de una imagen Ubuntu en Docker.
