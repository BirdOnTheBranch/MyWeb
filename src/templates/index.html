Sobre mi



Soy desarrollador back-end, vivo en Alicante y me apasiona programar.

Mi formación se ha basado en cursos de diferentes plataformas online como Udemy.com y codecademy.com, así como recibiendo clases individuales junto a muchas horas de prácticas. Cambie Netflix para buscar preguntas en Stackoverflow.

He desarrollado aplicaciones en Python desde hace dos años, usando los frameworks más conocidos, tanto Django como Flask. También uso Docker para facilitar la portabilidad y compatibilidad de mis proyectos, dedicando tiempo realizar buenas prácticas y código limpio. Manejo Linux, HTML, CSS y Javascript con solvencia.

Esta página a modo de CV es una aplicación Flask (v 1.1.1) que recibe datos desde las APIs de GitHub y Codewars. La app se ha desarrollado usando TDD y mocks en los tests. Se aloja en un dominio web de Pythonanywhere.com, aunque también es accesible como un repositorio público tanto en GitHub como en GitLab. Puedes clonar el repositorio y lanzarla localmente desde una imagen de Ubuntu en Docker, cuya configuración está incluida en el código fuente.

Este es mi algoritmo.

Proyecto Url-shortener

Recorta urls

Esta app permite al usuario introducir una dirección web en su página principal, donde podrá transformarla en un código de cinco dígitos que funcionara como la url original.

La aplicación programada en Django (v 3.0.2) guarda las urls y los códigos en una base de datos SQL y si el usuario lo desea, puede registrarse y administrar su página de perfil con las urls y códigos que él mismo haya almacenado. La app se puede gestionar a través del admin panel. Para el desarrollo se ha usado TDD y plantillas Bootstrap para el frontend.

Se puede ejecutar localmente en cualquier ordenador a través de una imagen Ubuntu en Docker.
